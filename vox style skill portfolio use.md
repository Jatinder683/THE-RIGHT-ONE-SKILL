############################################### PART 2 — ADVANCED ANIMATION & CAMERA LANGUAGE ######################################
####################################################################################################################################

Animation should never exist because software makes it possible.

Animation exists because reality contains movement.

The audience already understands gravity, inertia, momentum, friction, elasticity and weight from everyday life.

Your animation should obey those expectations.

Whenever animation violates reality, it should do so intentionally to communicate meaning, never because it "looks cool."

"I never noticed the animation."
